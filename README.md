# 🎂 Age Calculator (Servlet-Based Project)

A simple **Age Calculator Web Application** built using **Java Servlets and JSP**.

---



## 🚀 Purpose

This project was created to get **hands-on experience with Servlets** and understand how:

* Client → Servlet → JSP flow works
* Data is passed using `setAttribute()` / `getAttribute()`
* Request handling and internal routing works

It will also help me in building a **strong foundation for Spring Boot**, as Spring internally works on similar concepts.

> ⚠️ This is a **basic learning project**, not a major production-level application.

---

## 🌐 Live Demo

🔗 **Live Application:**
https://age-calculator-wwj4.onrender.com/

> Try it out — enter your date of birth and see your age instantly 🎂


## 🧩 Features

* 📅 Input Date of Birth
* 🎯 Calculates:

    * Age in **Years, Months, Days**
    * **Days remaining for next birthday**
* 🎉 Shows special message if **today is your birthday**
* ❌ Prevents selecting **future dates**

---

* Java Servlets
